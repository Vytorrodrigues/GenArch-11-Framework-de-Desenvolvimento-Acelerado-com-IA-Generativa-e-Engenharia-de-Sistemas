# GenArch-11-Framework-de-Desenvolvimento-Acelerado-com-IA-Generativa-e-Engenharia-de-Sistemas
📝O projeto GenArch-11 tem como objetivo explorar a sinergia entre a análise de sistemas estruturada e a automação via Inteligência Artificial Generativa. O escopo abrange o ciclo completo de desenvolvimento de 11 aplicações independentes, utilizando Python e Streamlit como stack principal.
